# 🎮 Tic-Tac-Toe GUI — Java Swing

A classic two-player **Tic-Tac-Toe** game with a clean, colorful graphical interface built entirely using Java Swing — no external dependencies required!

---

## ✨ Features

- 🔴 **Two-player gameplay** — Player X (red) vs Player O (blue)
- 🟩 **Win detection** — Highlights the winning combination in green
- 🤝 **Draw detection** — Automatically detects when the board is full
- 📊 **Score tracking** — Keeps track of wins and draws across multiple rounds
- 🔄 **New Game** — Resets the board while preserving scores
- 🗑️ **Reset Score** — Clears all stats and starts fresh
- 🚪 **Exit** — Close the app with one click
- 🖥️ **Centered window** — Launches neatly centered on your screen

## 🚀 How to Run

### Prerequisites
- Java JDK 11 or above installed
- Any Java IDE (IntelliJ IDEA recommended) or terminal

### Steps

**Using IntelliJ IDEA:**
1. Clone or download this repository
2. Open IntelliJ → `File → New Project` → choose Java
3. Place `TicTacToeGUI.java` inside the `src` folder
4. Right-click the file → `Run 'TicTacToeGUI.main()'`

**Using Terminal:**
```bash
# Compile
javac TicTacToeGUI.java

# Run
java TicTacToeGUI
```

---

## 🎯 How to Play

1. The game starts with **Player X's turn**
2. Click any empty cell to place your mark
3. Players alternate turns until someone wins or the board fills up
4. The winning row/column/diagonal is **highlighted in green**
5. Click **New Game** to play again!
